About

The App i have created is called NotesSaver, where you can save notes. When you first open the app you will start with no notes, so you press the "+" button to write your first note. You will be taken to a new activity where you can write a title and the content of the Note. Once you have finished writing your note you press the save button on the top right to save the note. You will be taken back to the main screen to view your notes. If you want to edit the note you press the note you want to edit and it will take you back into the edit Activity. If you want to delete the note, when you are in the editor simply press the delete button on the top right and it will show you a confirmation box and press yes The bell icon on the main activity set a reminder in 20 hours and send a notification for you to come back to the app.

Thank you for using my app!
